README

ABOUT
Recall-shift is a Max abstraction created by Jon Bellona (http://jpbellona.com). Recall shifting is a technique to sample and reexamine/replay input data within the last M samples of time. The abstraction was used as part of Great Speeches (2015) and discussed in Chapter 3 of Jon’s dissertation research.


INSTALLATION
To install, place the ‘recall-shift’ folder inside 'Max N > packages' directory. Then restart Max. 
You will then have access to the recall-shifter object.
